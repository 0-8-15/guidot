Network Setup

For your representative to be reachable on a mobile phone (or any other device not directly connected to the internet) some kind of tunnel service is required.

In principle this could be achieved among other options by a reflector service, virtual private network, a galic router (I2P) or an onion router (Tor). Future versions will probably support more of these options. Currently only onion routing is supported.

Setting Up Garlic Routing

On Android: I2P Setup

  1. Install I2P from play store, f-droid or elsewhere.
  2. Using the upper right menu configure a garlic client tunnel as SOCKS on port 9051 (FIXME, TDB: check, risky, half done.)
  3. Using the upper right menu configure a garlic server tunnel to listen forward to 127.0.0.1:7443. Give it any name you like.

Setting Up Onion Routing

On Android: Orbot Setup

Note: There where issues with Orbot up to the point that trying i2p seemed to be the better idea.

  1. Install Orbot from play store, f-droid or elsewhere.
  2. Using the upper right menu configure a onion service (a.k.a. "hidden service") to listen on tor port 443 and forward to port 7443. Give it any name you like.
  3. Stop and restart Orbot.
  4. Visit the onion service menu again. This time the service should be listet. Long-tap the entry and select copy to clipboard.
  5. Go back and paste the entry into the CN field. The CN field should not contain the service address (ending in .onion).

On Linux: Tor

For the setup of an onion service on Linux install tor and configure a onion service in torrc to listen on port 443 and forward to 127.0.0.1:7443.
